# Kotlin Weather CLI

Консольное приложение для получения информации о погоде.

## Функциональность
- Получение текущей погоды по городу
- Прогноз на несколько дней
- История запросов
- Симуляция API запросов

## Технологии
- Kotlin
- OOP принципы
- Sealed classes
- Data classes

## Запуск
```bash
kotlinc src/Main.kt -include-runtime -d WeatherApp.jar
java -jar WeatherApp.jar
```
